My only certainty is that with the "Clock Numerology" cipher I assigned a temporary value of 3600 to the letter 'A'. And then I think I multiplied 3 x 6 (not counting zeroes). And the rest is anyone's guess. An intersting thing I did for the last calculation was that I (think?) I added '1' to the first half of the alphabet, and then subtracted '1' from the last half of the alphabet.

About The "Clock Numerology" Cipher
